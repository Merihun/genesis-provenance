## Image Usage Guidelines

### Best Practices
1. Always use Next.js `Image` component for optimization
2. Include descriptive `alt` text for accessibility
3. Use `fill` prop with fixed aspect ratio containers
4. Add `priority` prop for above-the-fold images
5. Ensure proper aspect ratio matching

### File Locations
- All images stored in: `/public/`
- Reference in code as: `/image-name.jpg`
- Next.js automatically serves from public directory

### Image Sources
- All images sourced from professional photography
- High-quality luxury brand imagery
- Appropriate for commercial use
- Realistic, professional photography (no obvious AI artifacts)
